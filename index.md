# SiMuR - WearablePerMed Project
{: .fs-9 }

This repository provides technical information on the contributions of the [Multisensor Systems and Robotics Group, University of Oviedo](https://simur.dieecs.com/) to the WearablePerMed project.

It primarily documents Phase 2 activities, which focus on developing mathematical algorithms for large-scale data processing and pattern detection. We place an emphasis on software artifacts that may be valuable for this and other research initiatives.

The repository also includes research studies and technical reports related to the evaluation of the MATRIX wearable sensor and other project-related investigations.
