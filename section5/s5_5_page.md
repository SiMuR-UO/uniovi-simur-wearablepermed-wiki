# Semi-supervised Classification

This study explores HAR using a single wearable inertial sensor by combining labeled laboratory data and large volumes of unlabeled free-living data under semi-supervised learning frameworks.

The pipeline includes representation learning with a 1D convolutional autoencoder, supervised baselines, and semi-supervised strategies such as self-training, Self-Adaptive Thresholding (SAT), and FixMatch.

### Expected Benefits

- Leveraging unlabeled free-living data improves generalization.
- Representation learning helps capture motion patterns before classification.
- Semi-supervised techniques increase robustness while reducing annotation needs.
