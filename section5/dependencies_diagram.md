# Descripción
This diagram represent the artifacts relations. Actually we manage two types of articfacts:

- **Python Packages**: these artifacts are published inside [Simur Pypi Python Repository](https://pypi.org/manage/projects/).
- **Docker Images**: these artifacts are published inside [Simur Docker Hub Repository](https://hub.docker.com/repositories/simuruo).
